# Insurance Fraud Detection Using Machine Learning

## Project Overview
Insurance fraud is a major problem that causes significant financial loss to insurance companies. 
This project aims to build a machine learning model that predicts whether an insurance claim is fraudulent or genuine.

## Objectives
- Detect fraudulent insurance claims.
- Use machine learning algorithms to classify claims.
- Improve accuracy in fraud detection.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Machine Learning Models
- Logistic Regression
- Random Forest
- K-Nearest Neighbors (KNN)

## Project Workflow
1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis
4. Model Training
5. Model Evaluation
6. Fraud Prediction

## Result
The model predicts whether a claim is fraudulent or not using historical insurance claim data.
